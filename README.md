# Brand Smith Website

Static website for Brand Smith using the black, white, and neon visual language from the supplied logo.

## Open

Open `index.html` in a browser.

## Contact Email

The enquiry form currently opens the visitor's email app and sends to:

```js
hello@brandsmith.co
```

Change `CONTACT_EMAIL` in `script.js` if you want a different receiving address.

## Work Images

The panorama uses 10 work panels from `assets/`. Replace any image in that folder and update the matching `<img>` in `index.html` to use real portfolio images later.
